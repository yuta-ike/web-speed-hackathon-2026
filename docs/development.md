## 開発方法

### 必要なもの

- mise-en-place (mise)
  - https://mise.jdx.dev/

### セットアップ

1. mise を有効化します
   - ```bash
     mise trust
     ```
2. 依存パッケージをインストールします
   - ```bash
     mise install
     ```

## ディレクトリ構成

- `/application` : CaX のアプリケーションです
- `/scoring-tool` : CaX のパフォーマンス計測ツールです

開発方法・使用方法については、それぞれの README を参照してください。
